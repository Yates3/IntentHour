# Visual acceptance baseline

These five source images are the locked visual references for IntentHour v1:

1. `01-home-hero.png` — marketing hero and product preview.
2. `02-flow-privacy.png` — three-step flow and privacy boundary.
3. `03-pricing-footer.png` — lifetime pricing and site footer.
4. `04-focus-timer.png` — active focus workspace and interruption rail.
5. `05-weekly-review.png` — completed-week metrics, patterns, evidence, and export.

Implementation invariants: near-black graphite surfaces, amber primary actions, ice-cyan privacy/sync state, fine technical borders, tabular timer/metric numerals, restrained motion, and no glassmorphism, gamified scoring, chat interface, or cyberpunk decoration. Desktop and mobile browser screenshots should be reviewed beside these files before release.
